# 🧠 AI Quiz Game

An AI-powered Quiz Application built using **Python, Streamlit, Gemini AI, and Firebase Firestore**.

The application generates intelligent quiz questions, adapts difficulty based on user performance, tracks scores, and stores leaderboard data in real time.

---

## 🚀 Features

✅ AI-generated quiz questions using Gemini AI

✅ Multiple categories:

* Python
* AI
* Machine Learning
* DBMS
* Aptitude
* Cyber Security

✅ Adaptive difficulty system

* Easy
* Medium
* Hard

✅ Real-time score tracking

✅ Performance analytics

* Accuracy %
* Score trend graph
* Result distribution charts

✅ Firebase Firestore leaderboard

✅ Modern responsive UI

✅ Cloud deployment using Render

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/dhanujvardhan/AI-Quiz-Game.git
```

### Move into Project Folder

```bash
cd AI-Quiz-Game
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create the following environment variables:

### Gemini API Key

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Firebase Credentials

```env
FIREBASE_CREDENTIALS=YOUR_FIREBASE_SERVICE_ACCOUNT_JSON
```

---

## ▶️ Run Application

```bash
streamlit run app.py
```

---

## 📊 Application Workflow

1. Select a quiz category.
2. AI generates questions dynamically.
3. User submits answers.
4. Score and accuracy are updated.
5. Difficulty adapts automatically.
6. Performance analytics are displayed.
7. Final score is stored in Firebase Firestore.
8. Top players appear on the leaderboard.
